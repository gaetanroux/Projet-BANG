Le BANG est un jeu de cartes. Au départ, seul le shérif est connu. Les autres joueurs incarnent des adjoints au shérif, hors-la-loi ou renégat mais leur rôle est gardé secret.

Dans ce jeux vous trouverez 8 extentions. Vous aurez la possibilité de choisir votre extention.

- Bang ! : Armés et Dangereux ( Jeu de base )
- Bang! Gold Rush
- Bang! : Wild West Show
- Bang! : Dodge City
- Wanted ! : High Noon /A Fistful of Cards
- Wanted ! : Dodge City
- Bang! : Dodge City
- Bang! : High Noon
